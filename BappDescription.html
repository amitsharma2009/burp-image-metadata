This extension extract metadata present in image files. The information found is rarely critical, but it can be useful for general reconnaissance. These information can be usernames who created the files, local paths and technologies used.

Metadata type supported

  • JPEG Exif (using the library metadata-extractor)
  • PNG Text chunks (using the library PNGJ)

Usage

  • Go to: Proxy > History
  • Select a request that loads an image
  • The metadata can be found under: Response > Metadata
